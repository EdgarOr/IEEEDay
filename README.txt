Desarrollar una aplicación web de monitoreo administrable y la cual deberá insertarse en una base de datos
y poder visualizar toadas las alertas generadas. Administrable quiere decir que puede ajustar por ejemplo
el consumo de memoria del equipo. Puedo ajustar que me envíe una alerta cando se este consumiendo el 50% de la memoria, 
puedo cambiar ese valor de 50% a 90%, es por ello que se denomina administrable. 

++++ Extra: En caso de no haber conexión a la base de datos, la aplicación deberá guardar los datos en un archivo 
temporal para posteriormente insertarlos en la base de datos cuando tenga conexión nuevamente.
